## 函数
*一种功能的包装*
```
int add(int a,int b){
    int c =a +b;
    return c; //返回值要和函数的类型相同
}
int main(){
    int num =add(a,b);
    return0;
}
```
//函数就是把一类功能封装起来反复使用
