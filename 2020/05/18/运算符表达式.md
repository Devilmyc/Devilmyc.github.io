## 表达式
循环语句
```
#include <stdio.h>
int main(){
    int flag = 100;
    while(flag>=0){
        printf("2");
        flag = flag -1;
    }
    printf("1");
}
return 0;
```

    //在c语言除法中如果你的类型是整型结果会被截尾，如果是浮点型就会保留小数
    int a1 =13;
    int b1=5;
